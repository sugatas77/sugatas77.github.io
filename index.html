Introduction

A shiny app for predictive text demonstration. This is the capstone project of the Coursera Data Science program.

Data Source

The source for the data is from the HC Corpora which encompasses data from US blogs, news reports, and twitter messages. The data set is provided by Coursera as a ZIP file downloadable from their site.

Preparation and Prdiction

  • Download and Unzip The Data
  • Select US Data
  • Clean the Data
  • Tokenize the Data
  • Explore the Data & Sample the Data
  • Build N-Gram Model
  • Build Predictive Model

Data sampling was performed so as to reduce the size of the data.

App Data Processing and Performance

  1. User enters words to be completed.
  2. Code cleans and tokenizes the input words.
  3. Input words passed to prediction function.
  4. Prediction function searches bigram and trigram data for the top list of predicted words. This gives priority to trigram data over bigram
  5. Predicted words are displayed

The system runs little bit slow for the first run
